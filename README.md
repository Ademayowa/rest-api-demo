# REST API WITH GO, SQLITE AND GIN

## Getting Started

### Prerequisites

- Go 1.23+
- Gin
- SQLite

### Local Development

1. Clone the repository:

```bash
git clone https://github.com/Ademayowa/rest-api-demo
cd rest-api-demo
```

2. Install dependencies:

```bash
go mod download
```

3. Running the Project:

```bash
go run main.go
```
